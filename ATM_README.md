# ATM Menu Simulation 

A beginner-friendly Java program that simulates a simple ATM system.

##  Features
- Check Balance
- Deposit Money
- Withdraw Money
- Exit

##  Built With
- Java
- Control Structures (`if`, `switch`, `while`)
- Scanner for user input

##  Future Improvements
- PIN authentication
- Transaction history
- File saving/loading
- GUI with JavaFX or Swing
